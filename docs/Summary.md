## Depot Manager — Schedules

| Function | Description |
|----------|-------------|
| Approve pending trips | Trips submitted by scheduler/admin become visible to drivers in **My trips** |
| Reject pending trips | Return incomplete plans to the scheduler as drafts with a reason |
| Adjust live trips | Change times, bus, driver, status, and notes on approved trips |
| Review conflicts | Open the conflict panel and resolve overlaps in the adjust drawer |
| Emergency mode | Enable rapid operational changes when depot disruptions occur |

**Not available to Depot Manager:** Create timetable, send for approval, or delete schedules (scheduler/admin only).
